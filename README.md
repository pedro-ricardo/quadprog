# QuadProg Solver
Simple Quadratic programming solver. The solver credit goes to a `R` package.
This repo simply translate that solver to use a **Matlab** like interface.

* Both C and Fortran usage examples
* Free from memmory leaks

## Tested With:
| Library  | Version  |
| -------- | -------- |
| GCC      | 10.2.1   |
| Valgrind | 3.15.0   |
